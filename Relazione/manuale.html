Manuale d'uso

chance

Mostra la probabilità percentuale di infezione di un nodo.

contact

Mostra o aggiunge le informazioni di contatto tra due nodi.
Aggiunge un nuovo nodo se uno tra i contatti non si trova nel grafo.
Formato: "contact id-nodo1 id-nodo2" per mostrare le informazioni del contatto.
Formato: "contact id-nodo1 id-nodo2 distanza esposizione data" per aggiungere un nuovo contatto.

c

Scorciatoia per il comando contatto.

delcontact

Elimina un contatto.
Formato: "delcontact id-nodo1 id-nodo2"

d

Scorciatoia per delcontact.

exit

Chiude il programma.

help

Mostra la pagina di aiuto.

mark

Marca un nodo come infetto o pulito.
Formato: "mark id-nodo 0/1" 0 -> pulito, 1 -> infetto

node

Mostra o aggiunge le informazioni un nuovo nodo.
Formato: "node id-nodo" per mostrare le informazioni del un nodo.
Formato: "node id-nodo tipo" per aggiungere un nuovo nodo.
    Tipi di nodo disponibili:
  • 0 : Nodo generico
  • 1 : Persona
  • 2 : Edificio

load

Carica un file.
Formato: "load nome-file"

printall

Mostra la probabilità percentuale di infezione per tutti i nodi.

rule

Seleziona la regola.
Formato: "Rule indice" l'indice è 0, 1, 2..
    Tipi di regola disponibili:
  • 0 : Regola semplice
  • 1 : Regola complessa (calcolo con vere funzioni di probablità)
  • 2 : Regola per distanza (distanza minima da infetto)
  • 3 : Regola per azzerare i valori nel grafo

save

Salva su file.
Formato: "Save nome-file"
